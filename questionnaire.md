# HackOnVibe — Project Questionnaire

**1. What does your application/service do?**

AppPromoKit is an AI app-launch material generator. You enter an app name plus a one-line positioning statement, and it produces 5 channels of launch materials in seconds — landing-page copy, an App Store description with ASO keywords, tweets, a Reddit post, and a cold outreach email. A one-click "Export kit" downloads all channels as Markdown. It has a light/dark theme toggle and is bilingual (English / 中文), and is built with Next.js 16 / React 19 / Tailwind 4 / TypeScript on the DeepSeek v4 API.
